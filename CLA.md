## How to agree

Add this line to your pull request description, filling in your name and GitHub
username:

```
I have read and agree to the CLA (<Your Name>, @<username>).
```

A maintainer confirms the line before merging. For many contributors, enable a
CLA bot instead of checking by hand — e.g. **CLA Assistant**
(<https://cla-assistant.io>), which requires a one-time authorization by the
Project Owner and then records signatures on every pull request automatically.
